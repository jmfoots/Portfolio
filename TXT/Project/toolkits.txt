~Solid State Drive Validation Bash Tool|Winter 2018 - Fall 2019|https://jmfoots.github.io/Portfolio/PROJECTS/SSD/ProjectReport.pdf|IMG/ssd.png|
Click image to view project report PDF.
Created test data, files, and directory structures to run the other scripts against.
Simulated file alterations that a user could make while making a ‘best attempt’ to isolate these alterations to modify one metadata field at a time.
Created an image of a drive.
Setup the drive and its partitions for the creation of the file metadata files.
Created a File metadata csv used to display all possible ways a file could have been altered.
Created a Summary csv used to show that one or more partitions on the drive has changed and that the entire drive is not 100% the same.
Created a Comparison csv used to show that one or more files on the before/after file metadata csv files has changed, been deleted, or been added.

~KioskKit Powershell Tool|Fall 2019|https://jmfoots.github.io/Portfolio/PROJECTS/KKP/KioskKitProject.pdf|IMG/kkp.png|
Click image to view project report PDF.
Created an automation script to configure a new PC to be compliant with University of South Alabama Hardening Standard.
Configured standard user and admin user accounts.
Configured Group Policy setting using CIS-CAT.
Configured Local Security Policy to display University machine usage notice.
Configured Registry to automatically log on standard user.
Configured Task Scheduler to apply auto logon after logging into admin user.
Installed required programs and windows updates.
